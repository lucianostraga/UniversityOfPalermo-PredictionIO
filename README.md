## UniversityOfPalermo-PredictionIO
#Multinomial Naive Bayes (7 features implementation) for predicting drop-out rate at University of Palermo (Buenos Aires, Argentina)

#This engine supports training sets of seven variables, and bulk quieries as an array of JSONs with the attributes.

Also includes, python scripts for importing data, evaluating accuracy by PCA and t-SNE training set transformation, and traning set plotting.
Python scripts are based on Scikit Learn machine learning library.
